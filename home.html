About

This project compiles a clean list of Olympic host cities, geocoded for visualization. Data sources include Wikipedia, transformed via scripts in this repo. The map uses Leaflet with clustered markers and simple filters.

How to use

  • Open the Map to browse hosts by year and search by city/country.
  • Download CSV or JSON to analyze in your own tools.
  • Cancelled years (1916, 1940, 1944) are excluded from map markers.
